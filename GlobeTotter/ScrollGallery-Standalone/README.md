# Scroll Animation Gallery - React JSX

A beautiful scroll-triggered animation gallery built with React.

## ⚛️ **React JSX Version**

**Files:** 
- `ScrollGallery.jsx` (main component)
- `App.jsx` (app wrapper)
- `main.jsx` (entry point)
- `index.html` (React entry HTML)
- `package.json` (dependencies)
- `vite.config.js` (build config)

## 🚀 **How to Run:**

1. Open terminal in this folder
2. Run: `npm install`
3. Run: `npm run dev`
4. Opens at `http://localhost:3000`

## ✨ **Features**

- **Perfect scroll animations** - Items straight when centered, tilt when moving away
- **Lenis smooth scrolling** - Buttery smooth scroll experience
- **4 colored columns** - Gray, dark, colorful, vibrant
- **Center-based rotation** - Items are straight when at viewport center
- **Hover effects** - Items scale on mouse over
- **Mobile responsive** - 2 cols mobile, 4 cols desktop
- **Dark theme** - Black background with colored items
- **Minimal design** - Pure animated grid, no text
- **Optimized performance** - requestAnimationFrame for smooth animations
- **React hooks** - useState and useEffect for interactivity
- **Hot reload** - Changes update instantly during development

## 🎨 **Color Schemes**

- **Column 1**: Gray tones (#e1e1e1 to #868686)
- **Column 2**: Dark grays (#797979 to #1e1e1e)
- **Column 3**: Bright colors (#ff6b6b, #4ecdc4, #45b7d1, etc.)
- **Column 4**: Vibrant tones (#fd79a8, #fdcb6e, #e17055, etc.)

## 🔧 **Customization**

The component uses CSS-in-JS for styling, making it easy to customize colors, animations, and layout directly in the JSX file.

Perfect for integration into any React project! 🚀
